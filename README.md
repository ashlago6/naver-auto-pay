# Naver Pay Auto Clicker

뽐뿌 게시판에서 네이버페이 포인트 적립 링크를 자동으로 탐색하고 클릭하는 자동화 도구입니다.

---

## 목적

뽐뿌(ppomppu.co.kr) 게시판에 매일 올라오는 네이버페이 포인트 적립 이벤트 게시글을 자동으로 찾아 적립 링크를 클릭합니다.
수동으로 게시글을 하나씩 확인하고 링크를 클릭하는 반복 작업을 자동화합니다.

---

## 요구 사항

- Windows 10 / 11
- Python 3.11 이상
- 네이버 계정

---

## 설치

### 1. 저장소 다운로드

```
Code → Download ZIP
```

ZIP 압축을 풀거나 git clone 후 폴더로 이동합니다.

### 2. 환경 설치

`setup.bat`을 더블클릭하거나 CMD에서 실행합니다.

```bat
setup.bat
```

Python 패키지(`playwright`)와 Chromium 브라우저가 자동으로 설치됩니다.

---

## 최초 1회 — 네이버 로그인

프로그램이 네이버에 로그인된 상태로 동작하려면 세션을 먼저 저장해야 합니다.

```bat
login.bat
```

1. Chromium 브라우저가 열립니다.
2. 네이버 로그인 페이지에서 **직접 로그인**합니다.
3. 로그인 완료 후 **터미널 창으로 돌아와 Enter**를 누릅니다.
4. `session.json` 파일에 세션이 저장됩니다.

> 로그인 세션은 한 번만 저장하면 이후에는 자동으로 사용됩니다.
> 세션이 만료되면 `login.bat`을 다시 실행하세요.

## 실행

```bat
run.bat
```

### 날짜 범위 선택

실행 시 탐색할 날짜 범위를 선택합니다.

```
==================================================
  날짜 범위 선택
==================================================
  오늘: 2026-03-17

  1. 오늘만
  2. 오늘 ~ 1일 전
  3. 오늘 ~ 3일 전
  4. 오늘 ~ 1주일 전
  5. 직접 입력

  선택 (1~5):
```

| 옵션 | 탐색 범위 |
|------|-----------|
| 1 | 오늘 게시글만 |
| 2 | 어제 ~ 오늘 |
| 3 | 3일 전 ~ 오늘 |
| 4 | 1주일 전 ~ 오늘 |
| 5 | 날짜 직접 입력 (최대 7일 범위) |

### 동작 흐름

```
날짜 선택
  └→ 뽐뿌 검색 결과 페이지 순차 스캔
       └→ 키워드 매칭 게시글 수집
            └→ 각 게시글 본문에서 네이버페이 적립 링크 탐색
                 └→ 링크 클릭 → 참여 버튼 자동 클릭
                      └→ visited.json에 처리 기록 저장
```

탐색이 끝나면 계속/종료를 선택할 수 있습니다.

## 파일 구조

```
naver-auto-pay-master/
├── config.py        # 설정 (URL, 키워드, 딜레이, 보안 화이트리스트)
├── login.py         # 네이버 로그인 & 세션 저장
├── scraper.py       # 뽐뿌 게시판 크롤링
├── clicker.py       # 적립 링크 수집 및 클릭
├── main.py          # 메인 실행 파일
├── setup.bat        # 환경 설치
├── login.bat        # 로그인 단축 실행
├── run.bat          # 프로그램 실행
├── session.json     # 로그인 세션 (자동 생성)
└── visited.json     # 처리 완료 URL 목록 (자동 생성)
```

## 설정 변경

`config.py`에서 조정할 수 있습니다.

### 딜레이 (초 단위)

```python
DELAY = {
    "action_min": 2.0,  "action_max": 4.5,   # 클릭 사이
    "page_min":   3.0,  "page_max":   5.0,   # 페이지 이동 후
    "post_min":   3.0,  "post_max":   6.0,   # 게시글 열기 전
}
```

봇 차단 방지를 위해 `action_min` 1.0초 미만으로 낮추지 마세요.

### 최대 탐색 페이지 수

```python
BOARDS = [
    {
        "pages": 50,  # 날짜 범위 도달 시 자동 중단 (안전 상한선)
    }
]
```

## 주의 사항

- `session.json`은 네이버 로그인 정보가 담겨 있습니다. **외부에 공유하거나 git에 커밋하지 마세요.**
- 이미 클릭한 링크는 `visited.json`에 기록되어 중복 클릭하지 않습니다.
- 브라우저 창이 열린 상태로 동작합니다. 실행 중 브라우저를 강제로 닫지 마세요.
